Format XML

XML (Extensible Markup Language) to język znaczników do reprezentowania strukturalnych danych w formie tekstowej. Charakteryzuje się hierarchiczną strukturą opartą na znacznikach.

Format CSV

CSV (Comma-Separated Values) to prosty format przechowujący dane tabelaryczne w formie tekstu. Każdy rekord reprezentowany jest jako linia, a pola oddzielane są przecinkami.

Format JSON

JSON (JavaScript Object Notation) to lekki format wymiany danych, często używany w języku JavaScript. Jest czytelny dla ludzi i łatwo przyswajalny, a dane są reprezentowane w postaci obiektów.
